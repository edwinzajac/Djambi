## Structure

```mermaid
classDiagram
class AbstractBoard{
    boardMatrix 
        bool centerIsOccupied
        possibleNbPlayers=[]
        __init__()
        getPossibleMoves(self,piece,origincell)
        move(self,origincell,destcell)
}
class HexaBoard{possibleNbPlayers=[2,4]
    __init__(self)}
    class SquareBoard{
        __init__(self)
            possibleNbPlayers=[3,6]}
            class Game{
                int currentPlayer 
                    AbstractBoard gameBoard 
                    nextPlayer()
                    ...}
                    class Player{
                        string name
                            string color
                            int order
                    }
class Cell {
    int coordQ
        int coordS
        int coordR
        Piece piece
        isCenter()
}
class Piece {
    Image img
        string name
        string description
}
AbstractBoard <|-- HexaBoard
AbstractBoard <|-- SquareBoard

AbstractBoard "1" -- "1"Game
Game "1" -- "2..6" Player
AbstractBoard "1" -- "*" Cell
Cell "*" -- "1" Piece
```

## Make Commands

### Run game

```shell
make
```

### Install requirements 

```shell
make requirements
```

### Generate colorized assets 

```shell
make colorize-pieces 
```
